# EduPath 👥

> **Status do Projeto:** 🚀 MVP em Desenvolvimento / Planeamento de Integrações e IA  
> **Product Owner:** [Leandro Gasparello](https://linkedin.com/in/leandrogasparello)

O **EduPath** é uma plataforma de EdTech focada em aprendizagem personalizada e gestão de indicadores de treinamento para empresas e instituições de ensino. O produto centraliza a criação de trilhas de conhecimento, o acompanhamento de progresso e a análise de engajamento em tempo real.
